# Commit convention

When creating git commits, always use the following Co-Authored-By line:

```
Co-Authored-By: claude <168982+claude@users.noreply.github.com>
```

Do NOT use `Co-Authored-By: Claude Sonnet 4.6 <noreply@anthropic.com>` or `Co-Authored-By: claude <claude[bot]@users.noreply.github.com>`.
